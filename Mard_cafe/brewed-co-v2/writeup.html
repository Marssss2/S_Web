Website Title: MarD Brew Co. — Specialty Coffee Shop

Project Overview

Theme: Small Business — Specialty Coffee Shop  |  Pages: 5  |  Aesthetic: Light, Calming, Matcha Green & White

MarD Brew Co. is a fictional specialty coffee shop website built to demonstrate semantic HTML5 structuring, external CSS styling, responsive layout, and W3C-compliant web development. The color direction chosen was light, calming matcha green and white — using soft sage tones, warm cream backgrounds, and rounded cards to evoke a peaceful, welcoming atmosphere. Real photography was sourced from Unsplash (free, license-free) and embedded via direct URL to enhance visual appeal without requiring local image files.

Site Structure

  • index.html — Home: hero split layout, features, signature drinks, food preview
  • about.html — Brand story, values, team photos
  • menu.html — Drink image cards + 3 full pricing tables
  • services.html — Service cards + rates table + CTA
  • contact.html — Full inquiry form with all required input types
  • css/style.css — Single shared external stylesheet

HTML Design Choices

All pages use semantic HTML5 elements: <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer>. ARIA labels and roles are applied throughout for accessibility. A shared sticky navigation bar links all five pages with an active state indicator per page. The Contact page includes a full form with: text inputs, email, tel, <select> dropdowns, radio buttons (inquiry type), checkboxes (preferences), and a textarea — all properly labeled with <label for> and wrapped in <fieldset> with legends for screen reader support. Menu and Services pages use HTML <table> elements with <thead>, <tbody>, and scope attributes. Images are embedded using the <img> element with descriptive alt text on every image for accessibility compliance. Inner pages include a full-bleed background image in the page hero with an overlay.

CSS Design Choices

All styling is handled by one external stylesheet (css/style.css) linked to every page. CSS Custom Properties (:root variables) define the color system, typography, shadows, border radii, and transitions — ensuring consistency and easy theming. Layout uses CSS Flexbox (navbar, hero CTA, stats row, contact items, card footers) and CSS Grid (hero split, feature grids, menu cards, about section, contact two-column, footer). Responsive design uses media queries at 960px, 768px, and 540px — collapsing grid layouts, hiding desktop navigation, and stacking columns on mobile. CSS animations include: slideDown for the navbar, fadeUp for page content entry, and hover transitions with translateY, box-shadow changes, and transform: scale() on menu card images. A ::before pseudo-element sliding underline effect is applied to navigation links and service cards for micro-interaction polish. W3C standards followed: meaningful class names, proper specificity, comments, consistent indentation, and alt text on all images.

Media & Tools

Built with plain HTML5 and CSS3 — no JavaScript frameworks or CSS libraries. Google Fonts: Cormorant Garamond (display headings) and DM Sans (body text). All photography sourced from Unsplash (free commercial use license). Images are embedded via direct URL with width and quality parameters for optimized loading. The site is fully functional as a static local project opened in any modern browser.
